### Installation
Setup:

```sh
git clone https://github.com/suite/SupremeMonitor.git
cd SupremeMonitor
npm i # "sudo npm i" if you're on macOS or Linux
```

Config:
```sh
Edit config.js
```

Run After Setup:

```sh
node index.js
```
